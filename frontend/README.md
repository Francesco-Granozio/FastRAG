# RAG Application - Frontend

Frontend React + TypeScript per l'applicazione RAG.

## Tecnologie

- **React 19** - Libreria UI
- **TypeScript** - Type safety
- **Vite** - Build tool veloce
- **Tailwind CSS** - Styling utility-first
- **React Router** - Routing
- **TanStack Query** - Data fetching e caching
- **Axios** - HTTP client

## Setup

```bash
npm install
```

## Sviluppo

```bash
npm run dev
```

L'applicazione sarà disponibile su `http://localhost:3000`

## Build

```bash
npm run build
```

Il build sarà disponibile in `dist/` e può essere servito da FastAPI in produzione.

## Struttura

- `src/components/` - Componenti React
- `src/hooks/` - Custom hooks
- `src/services/` - API client e utilities
